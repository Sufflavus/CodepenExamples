<div class="page">
  <div class="introduction">
    The <a target="_blank" href="https://www.w3.org/wiki/HTML/Elements/input/range">HTML5 range input</a> element is styled according to the material <a target="_blank" href="https://material.io/guidelines/components/sliders.html">Slider spec</a>. Tested in IE, Opera, FF, Chrome, Safari. Unfortunately, so far (October 2017) there are no ways to fully style this element for webkit browsers (Chrome, Opera, Safari). Because of this, there is a JS (jQuery) code used for styling.
  </div>
  
  <h3>Normal</h3>
  <div class="range-wrapper">
    <input type="range" value="0"/>
  </div>
  <div class="range-wrapper">
    <input type="range" min="10" max="50" step="2" value="20"/>
  </div>
  <div class="range-wrapper">
    <input type="range" value="100"/>
  </div>
  
  <h3>Disabled</h3>
  <div class="range-wrapper">
    <input value="0" type="range"  disabled="disabled"/>
  </div>
  <div class="range-wrapper">
    <input type="range"  disabled="disabled"/>
  </div>
  <div class="range-wrapper">
    <input value="100" type="range"  disabled="disabled"/>
  </div>
</div>
